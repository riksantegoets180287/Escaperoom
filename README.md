# Virusvrij Escape Room

Een interactieve cybersecurity escape room voor leerlingen om op een leuke manier te leren over online veiligheid.

## Functies

- 6 verschillende mini-games over wachtwoorden, phishing, privacy en meer
- Voortgang wordt automatisch opgeslagen
- Certificaat bij voltooiing
- Admin panel voor het aanpassen van instellingen
- Volledig responsive design

## Aan de slag

De app start automatisch. Klik gewoon op "Start" om te beginnen!

### Admin toegang

Druk 5x op de toets '9' binnen 5 seconden om het admin panel te openen.

Standaard wachtwoord: `admin123`
